# 🪨 Stone Paper Scissors Game

A simple Stone Paper Scissors game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Play against the computer
- Random computer moves
- Live score tracking
- Reset game button
- Clean and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎮 How to Play

1. Click on Stone, Paper, or Scissors.
2. The computer makes a random choice.
3. The winner is displayed instantly.
4. Scores update automatically.
5. Click **Reset Game** to start over.
